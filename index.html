Welcome to LW3D

LW3D project is a lightweight 3D app template building on top of the configuration of:

  • SBT - http://www.scala-sbt.org
  • JME3 - http://jmonkeyengine.org
  • slf4j - http://www.slf4j.org
  • scalatest - http://www.scalatest.org

This project is maintained by Y-Experiment
